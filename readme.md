**Tsu** is a fullstack web boilerplate with a customizable stack, contained in a [Moon](https://github.com/moonrepo/moon) monorepo and deployable to a [Fly](https://fly.io/docs/machines/) machine as a [Docker](https://docs.docker.com/reference/dockerfile/) image. Frontend framework options are [SvelteKit](https://github.com/sveltejs/kit), (TBA). Backend framework options are [Elysia](https://github.com/elysiajs/elysia), (TBA). Frontend clients use [Lucide](https://github.com/lucide-icons/lucide/) icons, [Shadcn](https://github.com/shadcn-ui/ui) components, [Tailwind](https://github.com/tailwindlabs/tailwindcss) classes, (TBA).
- Install: `moon run :install`
- Spin up: `moon run :dev`
- Build and preview: `moon run :build`
