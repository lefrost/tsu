**Tsu** is a fullstack web boilerplate with a customizable stack, contained in [Moon](https://github.com/moonrepo/moon) and deployable to a [Fly machine](https://fly.io/docs/machines/) as a [Docker image](https://docs.docker.com/reference/dockerfile/). Frontend options are [SvelteKit](https://github.com/sveltejs/kit), (TBA). Backend options are [Elysia](https://github.com/elysiajs/elysia), (TBA).
- Install: `moon run :install`
- Spin up: `moon run :dev`
- Build and preview: `moon run :build`
